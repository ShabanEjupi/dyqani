# Enisi Center - Electronics & Smart Devices Website

## Overview

Enisi Center is a growing retail and distribution company based in Kosovo, specializing in high-quality consumer electronics and smart devices. This is our e-commerce website that allows customers to browse our electronics catalog, add products to their shopping cart, and proceed to checkout.

## Company Description

**Enisi Center** - A growing retail and distribution company based in Kosovo, specializing in high-quality consumer electronics and smart devices.

## Project Structure

```text
product-website
├── index.html          # Main entry point of the website
├── css
│   ├── style.css      # Custom styles for the website
│   └── normalize.css   # CSS reset for consistent styling
├── js
│   ├── main.js        # Main JavaScript file for website functionality
│   ├── products.js    # Handles loading and displaying product data
│   └── cart.js        # Manages shopping cart functionality
├── assets
│   └── icons
│       └── cart.svg   # SVG icon for the shopping cart
├── pages
│   ├── products.html   # HTML structure for the products page
│   └── checkout.html   # HTML structure for the checkout page
├── data
│   └── products.json   # Electronics product data in JSON format
└── README.md           # Documentation for the project
```

## Features

- Browse consumer electronics and smart devices with detailed specifications and prices
- Add electronics products to a shopping cart
- View and manage items in the cart
- Proceed to checkout to complete purchases
- Product categories include smartphones, audio devices, smart home, wearables, and accessories

## Setup Instructions

1. Clone the repository to your local machine
2. Open `index.html` in your web browser to view the website
3. Ensure that all assets and data files are correctly linked

## Usage Guidelines

- Navigate to the products page to view available electronics items
- Click on the "Add to Cart" button to add devices to your shopping cart
- Access the checkout page to review your cart and finalize your purchase
